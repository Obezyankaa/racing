- Game.js - главный оркестратор:

Инициализирует THREE.js renderer, сцену, камеру
Создаёт PhysicsWorld
Управляет game loop
Переключает сцены (парковка → трасса)

- PhysicsWorld.js:

Обёртка над Cannon.js/Rapier
Синхронизирует визуальные объекты с физическими телами
Обновляет физику в цикле

- SceneManager.js:

Загружает/выгружает сцены
Управляет переходами между парковкой и трассой
